Step one: pick a place for your scripts to go

Your scripts need a place to go on your computer or shared drive. The file path you indicate here will be added to each script to ensure a proper connection all around.

Pro tip: navigate to your shared drive location in file explorer and copy the path from there.

For example: Z:\BlueZone Scripts\

Step two: tell us some details about your county

Your county has county-specific information that you will need to use within scripts. The information you select here will be added to pertinent scripts.

For example: 003 for Anoka. This will be used frequently throughout scripts.
For example: OnBase, LaserFiche, SharePoint. Leave blank if no EDMS is used in your county. This will be used in places where a script must indicate where to store a document.
This is a local email address (someone in your county) who is designated to receive emails regarding scripts. It can be a single individual, or a distribution list (uses the "mailto:" command).

Step three: add your attorneys, magistrates, and judges

Several scripts need to use your attorneys, magistrates, and judges to fill out documents or other fields. Add your information below. If you don't know this information, stop, and discuss with a Child Support supervisor before proceeding.

Step four: advanced (optional) options

If you don't know what any of that means, skip this part.

Some counties may want to assign "beta users" to try out scripts using the same branch that scriptwriters use. Enter your beta users Windows logon IDs here, separated by commas. Do not add individuals here without consulting with the DHS scripts coordinator first.

Enter their logon IDs for Windows. For example, John F. Doe might be JFDOE. Each organization does this differently, so check with your user before assuming you know their IDs.

You should be all set! This will create a *.zip file containing all of your scripts, redirects, and a Power Pad. Download the file and extract it to the directory specified in the first step. Good luck!
